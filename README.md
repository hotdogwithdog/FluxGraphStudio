# FluxGraphStudio
Application for create Render graphs and apply them to images and videos. Basically a toy app for create and explore postprocessing techniques


# How to build
With CMake just do   
cmake --preset default

and inside the build folder that generates you will have the sln project

if you want to do with the CMakeGui i think that it will be just select the root folder for the CMakeList and use the preset default (not sure how to use presets on the GUI app)
